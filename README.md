# TCD 频谱图打标系统

经颅多普勒(TCD)频谱图标注系统，用于医学影像数据的标注、管理和分析。

## 项目简介

本系统是一个专为经颅多普勒超声频谱图设计的标注平台，支持以下功能：

- 频谱图浏览与管理
- 多维度标注（频谱形态、质量评估、置信度等）
- 高级筛选与过滤

## 技术栈

- 前端：React 19, Ant Design 5, Styled Components
- 后端：Node.js, Express
- 数据库：SQLite

## 项目结构
```
 label-system/
├── public/                 # 静态资源
│   └── images/             # 图片资源
│       └── spectrum/       # 频谱图存储位置（示例数据）
├── server/                 # 后端服务
│   ├── db/                 # 数据库相关（里面存储了初始化的数据库文件）
│   ├── routes/             # API路由
│   └── server.js           # 服务器入口
│   └── package.json        # 后端项目配置 （里面有后端数据库初始化和重置脚本）
├── src/                    # 前端源码
│   ├── components/         # 组件
│   ├── context/            # 上下文
│   ├── services/           # 服务层
│   ├── App.js              # 应用入口
│   └── index.js            # 渲染入口
├── .env                    # 环境变量
├── package.json            # 前端项目配置
└── README.md               # 项目说明
```



## 需要修改的部分
- 前端：
快捷键可以自行全局搜索关键词“快捷键”进行修改
- 后端：
  需要在`server.js`中修改数据库频谱图文件路径（按照自己实际情况修改），否则频谱图无法显示

  ```
  const IMAGE_BASE_PATH = 'E:/DB'; // 您的本地图片根目录
  ```

## 安装与启动 
双击运行`start.bat`即可
注意启动前端和后端的两个窗口不要关闭（应该是最后两个窗口）

## 注意事项
在标注时，注意关闭中文输入法，以免快捷键无法使用。
